== Running

This starter runs via *SpringBoot* using the `maven` command:

 
`mvn spring-boot:run` *or* 

`mvn clean package && java -jar target/banking.war` *or*

`./run.sh/run.cmd`


The application will be available at http://localhost:8080/banking
